# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

Please report security vulnerabilities via GitHub Security Advisories:
https://github.com/jagkaransingh/mern-ecommerce/security/advisories/new

Please do NOT report vulnerabilities through public GitHub issues.

You should receive a response within 48 hours. If the issue is confirmed,
we will release a patch as soon as possible.
